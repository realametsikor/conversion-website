How to Use

  1. Click "Select GIF" or drag and drop a GIF file into the workspace above.
  2. View the original file size and dimensions displayed after upload.
  3. Adjust compression options: scale down dimensions, reduce the number of colors, or skip frames to lower file size.
  4. Click "Compress GIF" to start the compression process.
  5. Review the compressed size and savings percentage, then click "Download Compressed GIF" to save the result.
  6. Optionally, click any extracted frame to download it as a standalone PNG image.

Frequently Asked Questions

Are my GIF files secure?

Yes. This tool processes GIF files entirely in your browser using the HTML5 Canvas API. Your files are never uploaded to any server and never leave your device.

How does the compression work?

The tool extracts individual frames from your GIF, optionally scales them down, reduces the color palette, and removes intermediate frames. The remaining frames are re-encoded into a new, smaller GIF file.

How much can the file size be reduced?

Typical reductions range from 30% to 70% depending on the original GIF and the options selected. Scaling down to 50% combined with frame skipping produces the largest savings.

Will the animation still look smooth?

Skipping frames removes some in-between frames, which may make the animation slightly less smooth. Keeping every 2nd frame is a good balance between size and smoothness. The frame delay is adjusted to maintain similar playback speed.

Can I download individual frames?

Yes. After compression, all extracted frames are displayed in a grid. Click any frame to download it as a high-quality PNG image.
