How to Use

  1. Click "Select Images" to upload multiple JPG, PNG, or WebP photos from your device.
  2. Choose a grid layout — 2x1, 2x2, 3x1, 1x3, or 3x3 — depending on how many images you want in your collage.
  3. Use the spacing slider to adjust the gap between images (0–20 pixels).
  4. Pick a background color for the gaps and borders using the color picker.
  5. Preview your collage in real time, then click "Download Collage as PNG" to save it.

Frequently Asked Questions

What size is the output collage?

The collage canvas is 1200 pixels wide. The height is calculated automatically based on the grid layout you choose, so the aspect ratio stays balanced for every configuration.

How many images can I add to a collage?

The number of images depends on the layout you select. A 2x1 layout uses 2 images, 2x2 uses 4, 3x1 and 1x3 use 3, and a 3x3 layout uses up to 9 images. Extra images beyond the grid size are ignored.

Are my photos uploaded to a server?

No. All processing happens entirely in your browser using the HTML5 Canvas API. Your photos never leave your device, so your privacy is fully protected.

What format is the downloaded collage?

The collage is downloaded as a high-quality PNG file, which preserves sharp details and supports transparency if your background is set to a solid color.

What happens if I upload fewer images than the grid requires?

Empty cells will be filled with the selected background color. You can still download the collage — it just means some grid slots will appear blank.
