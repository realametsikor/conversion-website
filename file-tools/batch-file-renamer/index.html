How to Batch Rename Files

  1. Click "Select Files" or drag and drop multiple files into the upload area above.
  2. Configure your renaming options: add prefixes, suffixes, find and replace text, or enable sequential numbering. The preview table updates live.
  3. Click "Download All as ZIP" to download all files with their new names in a single ZIP archive.

Frequently Asked Questions

Can I combine multiple renaming options?

Yes! All renaming options work together. For example, you can find and replace text while also adding a prefix and suffix. The operations are applied in order: find/replace first, then prefix, then suffix, then sequential numbering if enabled.

How does sequential numbering work?

When sequential numbering is enabled, each file name (before the extension) is replaced with a numbered format like file_001, file_002, etc. You can set the starting number. The prefix and suffix options still apply around the numbered name.

Are my files uploaded to a server?

No. All processing happens entirely in your browser. Your files are read locally, renamed, and packaged into a ZIP archive using JavaScript. Nothing is ever sent to a server.

Is there a limit on the number of files?

There is no strict limit, but processing a very large number of files (hundreds) or very large files may be slower depending on your device's memory and processing power.

What file types are supported?

Any file type is supported. The tool renames files regardless of their format. The file extension is preserved during renaming to ensure files remain functional.
